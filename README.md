# HR-Analytics-Dashboard-Workforce-Insights-Performance-Metrics

## Project Overview
This interactive HR dashboard provides comprehensive insights into workforce demographics, performance metrics, and organizational structure. The visualization enables HR professionals and management to make data-driven decisions about talent management, compensation strategies, and workforce planning.

## Interactive Dashboard
For the full interactive experience, please visit the dashboard on Tableau Public:
[View HR Analytics Dashboard][(https://public.tableau.com/app/profile/your-username/viz/HR-Analytics-Dashboard)](https://public.tableau.com/app/profile/jie.liang3815/viz/HRDashboard_17409029486550/HRSummary?publish=yes)

## Key Features
- Employee demographics breakdown by gender, education, and age
- Compensation analysis across different education levels and roles
- Department-wise employee distribution
- Performance metrics visualization
- Geographic distribution of workforce

## How to Use
The dashboard is fully interactive. You can:
- Hover over data points to see detailed information
- Use filters to focus on specific segments
- Click on elements to drill down into more specific data

## Data Source
This visualization was created using anonymized HR data. All metrics and insights represent typical organizational patterns while maintaining confidentiality.
